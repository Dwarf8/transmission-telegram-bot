## Available commands
* <b>List of all features</b>;
* <b>Torrent info</b> - Get all details about a torrent;
* <b>Torrent control</b> - Start, stop, delete torrents;
* <b>Add torrent</b> - Add a torrent from a <code>.torrent</code> file or from magnet url;
